Detailed Description

A future (pending) result of a Pubnub transaction/operation/request.

It is somewhat similar to the std::future<> from C++11.

It has the same interface for both Pubnub C client's "sync" and "callback" interfaces, so the C++ user code is always the same, you just select the "back-end" during the build.
